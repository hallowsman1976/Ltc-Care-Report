# 📱 คู่มือผู้ดูแล (Caregiver) — ฉบับย่อ
## ระบบ LTC Care Report

> สำหรับ อสม. / Caregiver (CG) ที่ลงพื้นที่เยี่ยมบ้าน · ใช้งานบนมือถือได้เลย

---

## 🔑 เข้าสู่ระบบ
1. เปิดลิงก์ระบบที่ได้รับจากผู้ดูแล (บันทึกหน้าจอ/Add to Home Screen ไว้ใช้ง่าย)
2. กรอก **ชื่อผู้ใช้ / รหัสผ่าน** ที่ได้รับ → กด **เข้าสู่ระบบ**
3. ครั้งแรกควรเปลี่ยนรหัสผ่าน (เมนู ⚙️ ตั้งค่า → โปรไฟล์)

> 🔒 ท่านจะเห็น **เฉพาะผู้รับบริการที่อยู่ในความรับผิดชอบของท่าน** เท่านั้น

---

## 🏠 บันทึกการเยี่ยมบ้าน (งานหลัก)
1. แตะเมนูล่าง **🚪 เยี่ยม**
2. **เลือกผู้รับบริการ** (พิมพ์ชื่อค้นหา)
3. ดูสรุปข้อมูลผู้ป่วย → ระบบใส่ **ครั้งที่เยี่ยม** ให้อัตโนมัติ
4. กรอก:
   - วันที่ / เวลาเริ่ม-สิ้นสุด
   - **อาการหลัก** (จำเป็น)
   - การดูแลที่ให้ / สุขศึกษา / การมีส่วนร่วมของครอบครัว
   - ปัญหาที่พบ (สิ่งแวดล้อม/ยา/อาหาร/ฟื้นฟู/จิตสังคม)
   - การส่งต่อ + **นัดเยี่ยมครั้งถัดไป**
5. **แนบรูป** (ถ้ามี): ✅ ติ๊กยินยอม PDPA → เลือกรูป (jpg/png/webp)
6. กด **💾 บันทึกการเยี่ยม** → ยืนยัน

---

## 📋 ทำแบบประเมิน (5 ขั้น)
แตะเมนู **📋 ประเมิน** → เลือกผู้ป่วย + การเยี่ยม → ทำทีละขั้น (ข้ามได้):

| ขั้น | ทำอะไร |
|-----|--------|
| 1️⃣ Vital Signs | น้ำหนัก/ส่วนสูง (BMI คิดให้) · ความดัน · DTX · O2 — ระบบแปลผลให้ |
| 2️⃣ ADL | กดคะแนน 10 ข้อ — ระบบรวมคะแนน + บอกกลุ่ม TAI · **ADL ≤ 11 = พึ่งพิง** |
| 3️⃣ PPS | เลือก % สำหรับผู้ป่วยระยะท้าย |
| 4️⃣ สุขภาพจิต | ตอบ 2Q ก่อน → ถ้าเสี่ยงจะให้ทำ 9Q ต่อ |
| 5️⃣ บริการ LTC | เลือกบริการที่ให้ในเดือนนี้ |

> 🚨 **สำคัญ:** ถ้าประเมิน 9Q แล้วผู้ป่วยมี**ความคิดทำร้ายตนเอง** ระบบจะขึ้น**เตือนสีแดง**
> → โทร **สายด่วนสุขภาพจิต 1323** ทันที + แจ้งครอบครัวให้เฝ้าระวัง + แจ้ง CM

> ⚠️ ผลประเมินเป็นการ **คัดกรองเบื้องต้น** ไม่ใช่การวินิจฉัย — ส่งต่อแพทย์เมื่อพบความผิดปกติ

---

## 🗺️ ดูแผนที่เคส
แตะเมนู **🗺️ แผนที่** — หมุดสีบอกสถานะ:
- 🟢 เขียว = ดูแลตามแผน
- 🟡 เหลือง = ใกล้ครบกำหนดเยี่ยม
- 🔴 แดง = เลยกำหนด/ต้องรีบติดตาม
- 🟣 ม่วง = Palliative Care

แตะหมุด → ดูข้อมูล + กด **🧭 นำทาง** เปิด Google Maps ไปบ้านผู้ป่วยได้เลย

---

## 💡 เคล็ดลับ
- เพิ่มพิกัด GPS ตอนอยู่ที่บ้านผู้ป่วย (ปุ่ม 📍) เพื่อให้แผนที่แม่นยำ
- ตั้ง **นัดครั้งถัดไป** ทุกครั้ง เพื่อให้ระบบเตือนเมื่อใกล้/เลยกำหนด
- ถ่ายรูปแผล/สภาพแวดล้อมแนบไว้เป็นหลักฐานการดูแล (ขอความยินยอมก่อนเสมอ)
- มีปัญหา → ติดต่อ Care Manager (CM) หรือผู้ดูแลระบบ

---

## ❓ ปัญหาที่พบบ่อย
| ปัญหา | วิธีแก้ |
|-------|--------|
| เข้าระบบไม่ได้ | ตรวจชื่อผู้ใช้/รหัส · ติดต่อผู้ดูแลระบบ |
| ไม่เห็นผู้ป่วยที่ต้องการ | ผู้ป่วยอาจไม่ได้อยู่ในความรับผิดชอบของท่าน — แจ้ง CM |
| แนบรูปไม่ได้ | ต้องติ๊กยินยอม PDPA ก่อน · ใช้ไฟล์ jpg/png/webp |
| บันทึกไม่ได้ | กรอกช่องจำเป็น (มี *) ให้ครบ · ตรวจสัญญาณเน็ต |
